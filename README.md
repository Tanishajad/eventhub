# EventHub – College Event Management System

EventHub is a DBMS-based web application that allows students to discover, book, and manage college events easily.  
It also provides an admin dashboard to manage events, users, bookings, and queries.

## Features

- User registration and login
- Browse upcoming college events
- Search events
- Book events
- Cancel bookings
- Contact form for queries
- Admin dashboard
- Add / Edit / Delete events
- Manage users
- Manage bookings
- Reply to student queries

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP (Apache Server)


## Database

The system uses a MySQL database with tables such as:

- users
- events
- bookings
- contact_queries
